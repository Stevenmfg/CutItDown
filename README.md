# Cut It Down
短链接生成应用  v1.1.1

这是我做的第一个应用，专注于短链接生成，API可以选择调用[新浪](http://open.weibo.com/wiki/2/short_url/shorten)、[百度](http://dwz.cn/)、[985.so](http://985.so/)还有[is.gd](http://is.gd/),
后续会根据需求添加更多可选的API。
